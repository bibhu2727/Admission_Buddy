# AdmBuddy - Admission Lead Manager

Lightweight, fast, PWA lead management tool for admission counselors.

## Features
- Email/Password login (Admin & Counselor roles)
- Lead data entry (Name, Phone, Channel, Remark, Visit Date)
- Today's Visits highlighted on dashboard
- Search by name or phone
- Lead status tracking: Pending, Converted, No Show, Postponed
- Progressive Web App (installable on phone)

## Tech Stack
- Next.js 16 (App Router)
- MongoDB Atlas
- NextAuth.js
- Tailwind CSS v4
